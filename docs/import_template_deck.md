# Добавление и импорт шаблонных баз (колод) через Artisan

Для автоматизации добавления новых колод и исключения ошибок форматирования создана специальная Laravel Artisan-команда. Она позволяет создавать папки, наполнять колоды карточками (из файлов или напрямую из консоли) и автоматически регистрировать новые категории в `categories.php`.

---

## Использование команды

Запустите команду в Docker-контейнере:

```bash
docker compose exec php php artisan deck:import-template
```

### Шаги работы команды:
1. **Выбор языка:** Выберите язык интерфейса (`ru`, `en`, `uk`).
2. **Категория:** Выберите одну из существующих категорий или выберите `[Create New Category]` для создания новой категории.
3. **Подкатегория:** Выберите существующую подкатегорию или `[Create New Subcategory]` для создания новой. При создании новой категории или подкатегории они автоматически дописываются в `categories.php` соответствующего языка с сохранением синтаксиса.
4. **Метаданные колоды:**
   - Имя файла колоды (Slug, без `.php`, например: `travel_phrases`).
   - Название колоды (например: `✈️ Фразы для путешествий`).
   - Краткое описание колоды.
   - Язык вопросов и язык ответов (по умолчанию подставляются языковые коды).
   - Порядок сортировки `sort_order` (опционально).
5. **Импорт карточек:**
   - **From a local text file:** Укажите путь к текстовому файлу и разделитель (по умолчанию `-`). Каждая строка вида `Вопрос - Ответ` превратится в карточку.
   - **Paste raw text in CLI:** Позволяет скопировать и вставить строки с карточками прямо в терминал. Для окончания ввода введите `DONE` на новой строке.
   - **Empty template:** Создает файл с 2 демонстрационными карточками для ручного заполнения.
6. **Сидирование БД:** Команда спросит, нужно ли запустить `TemplateDeckSeeder`, чтобы применить изменения в базу данных. При согласии будет запущен `php artisan db:seed --class=TemplateDeckSeeder`.

## Команда безопасного обновления базы (`my:fresh`)

Для быстрого и безопасного обновления структуры и наполнения шаблонов (без риска потерять пользователей и их прогресс) используйте команду:

```bash
docker compose exec php php artisan my:fresh
```

### Что она делает:
1. **Безопасная накатка миграций:** Запускает `php artisan migrate` (применяет новые миграции, не удаляя существующие данные).
2. **Обновление шаблонов:** Запускает `php artisan db:seed --class=Database\Seeders\TemplateDeckSeeder`, которая считывает файлы из `database/seeders/seeds_data` и обновляет шаблоны колод.
3. Команда **НЕ** запускает общие сидеры вроде `UsersTableSeeder`, сохраняя всех существующих пользователей.

## Устранение возможных проблем

### База данных выдает ошибку Foreign Key:
При создании колоды ее ID категории должен присутствовать в таблице `template_categories`. Команда `deck:import-template` автоматически записывает категорию в `categories.php` перед генерацией файла колоды. Если вы запускаете седер вручную, убедитесь, что категория сначала прописана в `categories.php`.
